## 10. YES Initiative (Level Boost — NOT Points)

YES is NOT part of any scored total. It provides level improvement:

| Achievement | Level Improvement |
|------------|------------------|
| Meet YES target + 2.5% absorption | 1 level up |
| Double YES target + 5% absorption | 2 levels up |
| 1.5x YES target + 5% absorption | 1 level up + 3 bonus points |

### YES Qualification Criteria (Slide 123)
- Met 40% sub-minimum on ALL priority elements, OR
- Average 50% across all priority elements (35 out of 70 points)

## 15. NPAT Deemed Calculation (Slide 24)

If NPAT margin < 25% of industry norm:
1. Calculate Indicative Profit Margin (IPM) = NPAT / Turnover
2. Use most recent NPAT (not older than 5 years) that is >= 25% of industry norm
3. New Target = Element target % x IPM (actual, not 25%) x Turnover
4. If no valid NPAT within 5 years: target becomes 25% of latest industry norm

## 18. Critical Bugs to Never Reintroduce

1. **Executive Directors targets**: Must be 50% Black / 25% Women (NOT 60% / 30%)
2. **Procurement BO51**: Must be 50% target for 11 points (NOT 40% for 10 points)
3. **Procurement DG**: Must be 2% target (NOT 12%)
4. **Procurement bonuses**: Do NOT exist (bonuses are ED only)
5. **Skills default target**: Must be 3.5% (NOT 6%)
6. **Skills absorption**: Must use `isAbsorbed` field (NOT `isEmployed`)
7. **Grand total**: Must be 120 for RCOGP Generic (NOT 111, NOT 121, NOT 132)
8. **MC total**: Must be 19 combined for RCOGP (NOT split as 8+11)
9. **ESD split**: Must be SD=10 + ED=7 for RCOGP (NOT combined 15)
10. **YES Initiative**: Is a level boost, NOT scored points. maxPoints must be 0.
11. **ICT Generic ED**: Must be 18 (NOT 15), total 140
12. **FSC Generic Skills**: Must be 23 (NOT 20)
13. **FSC Generic PP**: Must be 24 (NOT 20)
14. **ICT Level thresholds**: L1=120, L8=55 (NOT standard 100/40)
15. **FSC Level thresholds**: Scaled (95.5, 90.7, ...) (NOT standard integers)
